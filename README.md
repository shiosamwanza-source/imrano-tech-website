## ⚡ Solar Calculator Logic

The built-in solar calculator uses Tanzania-specific parameters:
- **4 peak sun hours/day** (Tanzania average)
- **25% efficiency loss buffer**
- **2 days battery autonomy**
- **50% depth of discharge (DoD)**
- Auto-recommendation by system size (small/medium/large/commercial)
